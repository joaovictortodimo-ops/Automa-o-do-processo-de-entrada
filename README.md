# Automação processo de-entrada
O sistema, baseado em prompt, realiza a conferência automática  Se estiver tudo correto → a entrada é registrada no estoque  Se houver erro → o sistema envia um alerta para o setor responsável  O sistema gera automaticamente:  Protocolo de recebimento  Local de armazenamento  Relatório de entrada  O material é direcionado para o local indicado.![IMG_3625](https://github.com/user-attachments/assets/33a46807-c255-40ad-b276-76477be36eeb)
